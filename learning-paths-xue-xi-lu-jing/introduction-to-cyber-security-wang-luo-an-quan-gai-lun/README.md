# Introduction to Cyber Security(网络安全概论)

网络安全是一个庞大的话题，可能很难知道从哪里开始。

这条路径将为你提供关于网络不同领域的实际介绍，包括:

* 网络安全职业。
* 进攻性安全；尝试攻击第一个应用程序(模拟)。
* 防御性安全；防御实时网络攻击。
* 探索行业中的安全主题。

完成这条学习路径将为开启你的网络之旅提供知识帮助。

先决条件：

开始这条路径不需要任何先决条件!仅需要拥有学习热情!
